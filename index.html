名刺管理アプリ

GitHub Pages + Supabase でどのデバイスからでも使える名刺管理アプリです。初回のみSupabaseの接続情報を設定してください。

Supabase セットアップ手順：
1. supabase.com でプロジェクトを作成
2. SQL Editorで下のテーブルを作成
3. Project URL と anon key を入力
SQLテーブル作成コード（クリックして展開） 
create table meishi (
  id uuid default gen_random_uuid() primary key,
  name text,
  company text,
  title text,
  email text,
  phone text,
  address text,
  tags text[],
  created_at timestamptz default now()
);
-- 認証なしで読み書きできるようにする（個人利用向け）
alter table meishi enable row level security;
create policy "allow all" on meishi for all using (true) with check (true);

入力した情報はブラウザのlocalStorageに保存されます
